### webpack.config.js 번들링 환경 설정하기

1. 설치 패키지
    - ```npm i -D webpack webpack-cli express```

2. 번들링 환경 설정(webpack.config.js)
    - entry
    - output

3. 스크립팅
```json
"scripts": {
    "build": "npx webpack",
    "start": "node dev-server"
  }
```

4. 빌드
    - $ npm run build

5. 실행
    - $ npm start
